# 项目代码修改方案

## 1. 修改概览

本次修改为 `Sidebar` 组件添加了**编辑会话标题**的功能，用户可以直接在侧边栏中编辑并保存会话标题。

**涉及文件：**
- `frontend/hooks/useSessions.ts` - 添加 `updateSessionTitle` 函数
- `frontend/components/layout/Sidebar.tsx` - 添加编辑标题的 UI 和逻辑
- `frontend/components/layout/AppLayout.tsx` - 传递 `onUpdateSessionTitle` 属性
- `frontend/App.tsx` - 连接 `updateSessionTitle` 函数

---

## 2. 文件修改详情

### 修改文件：`frontend/hooks/useSessions.ts`

**修改说明：**
添加了 `updateSessionTitle` 函数，用于更新会话标题并保存到数据库。

**关键代码：**
```typescript
const updateSessionTitle = useCallback((sessionId: string, newTitle: string) => {
  setSessions(prev => {
    const updated = prev.map(s => {
      if (s.id === sessionId) {
        const updatedSession = { ...s, title: newTitle };
        saveSessionToDb(updatedSession);
        return updatedSession;
      }
      return s;
    });
    return updated;
  });
}, [saveSessionToDb]);
```

### 修改文件：`frontend/components/layout/Sidebar.tsx`

**修改说明：**
1. 添加了编辑状态管理（`editingSessionId`、`editingTitle`）
2. 添加了编辑按钮（铅笔图标）
3. 添加了编辑模式的输入框和保存/取消按钮
4. 支持键盘快捷键（Enter 保存，Escape 取消）

**关键功能：**
- 点击编辑按钮进入编辑模式
- 输入新标题
- 按 Enter 或点击勾号保存
- 按 Escape 或点击 X 取消

### 修改文件：`frontend/components/layout/AppLayout.tsx`

**修改说明：**
在接口中添加了 `onUpdateSessionTitle` 属性，并传递给 `Sidebar` 组件。

### 修改文件：`frontend/App.tsx`

**修改说明：**
从 `useSessions` 中获取 `updateSessionTitle` 函数，并传递给 `AppLayout` 组件。

## 3. 功能特性

### 编辑会话标题

1. **触发编辑：**
   - 鼠标悬停在会话上
   - 点击铅笔图标（Edit2）

2. **编辑模式：**
   - 输入框自动聚焦
   - 显示当前标题
   - 可以输入新标题

3. **保存标题：**
   - 按 Enter 键
   - 点击勾号按钮（Check）
   - 自动保存到数据库

4. **取消编辑：**
   - 按 Escape 键
   - 点击 X 按钮
   - 恢复原标题

5. **UI 反馈：**
   - 编辑模式下显示蓝色边框
   - 保存按钮显示绿色
   - 取消按钮显示灰色
