# Virtual Try-On 改进方案

## 当前问题分析

### 1. 缺少掩码可视化

**问题**：
- 用户无法看到 AI 识别的服装区域
- 无法确认分割是否准确
- 无法调整分割区域

**影响**：
- 用户体验不完整
- 无法进行精确控制
- 与原始 Notebook 流程不一致

### 2. 工作流程不完整

**原始 Notebook 流程**：
```
1. 上传图片
2. Gemini 2.5 分割 → 生成掩码
3. 可视化掩码（用户确认）
4. Imagen 3 编辑 → 生成结果
```

**当前实现流程**：
```
1. 上传图片
2. 描述服装
3. 直接生成结果（跳过了掩码可视化）
```

#### 实现要点

1. **添加掩码生成 API**：
```typescript
// frontend/services/providers/google/media/virtual-tryon.ts

export const generateSegmentationMask = async (
  image: Attachment,
  targetClothing: string,  // 'upper', 'lower', 'full'
  apiKey: string
): Promise<{ maskUrl: string; segmentationData: SegmentationResult[] }> => {
  // 1. 调用 Gemini 2.5 分割
  const segmentationData = await segmentClothing(ai, image, targetClothing);
  
  // 2. 生成掩码图像
  const maskBlob = await generateMaskAsync(image, segmentationData);
  
  // 3. 创建 Blob URL
  const maskUrl = URL.createObjectURL(maskBlob);
  
  return { maskUrl, segmentationData };
};
```

2. **添加掩码预览组件**：

## 技术实现细节

### 1. 掩码生成 API

**后端路由**（已实现）：
```python
# backend/app/routers/tryon.py

@router.post("/segment")
async def segment_clothing(request: SegmentRequest):
    """
    生成服装分割掩码
    
    Returns:
        {
            "mask_base64": "data:image/png;base64,...",
            "segmentation_data": [...]
        }
    """
    pass
```

**前端调用**：
```typescript
// frontend/services/providers/google/media/virtual-tryon.ts

export const generateMaskPreview = async (
  imageBase64: string,
  targetClothing: string,
  apiKey: string
): Promise<string> => {
  const response = await fetch('/api/tryon/segment', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      image_base64: imageBase64,
      target_clothing: targetClothing,
      api_key: apiKey
    })
  });
  
  const data = await response.json();
  return data.mask_base64;  // data:image/png;base64,...
};
```

### 3. 状态管理

```typescript
// VirtualTryOnView.tsx

interface MaskState {
  maskUrl: string | null;
  segmentationData: SegmentationResult[] | null;
  isGenerating: boolean;
  showPreview: boolean;
}

const [maskState, setMaskState] = useState<MaskState>({
  maskUrl: null,
  segmentationData: null,
  isGenerating: false,
  showPreview: false
});

// 生成掩码
const handleGenerateMask = async () => {
  setMaskState(prev => ({ ...prev, isGenerating: true }));
  
  try {
    const maskUrl = await generateMaskPreview(
      activeImageUrl,
      options.virtualTryOnTarget,
      apiKey
    );
    
    setMaskState({
      maskUrl,
      segmentationData: null,  // 可选
      isGenerating: false,
      showPreview: true
    });
  } catch (error) {
    console.error('掩码生成失败:', error);
    setMaskState(prev => ({ ...prev, isGenerating: false }));
  }
};

// 确认掩码
const handleConfirmMask = () => {
  setMaskState(prev => ({ ...prev, showPreview: false }));
  // 继续到服装描述步骤
};

// 重新生成
const handleRegenerateMask = () => {
  handleGenerateMask();
};
```
