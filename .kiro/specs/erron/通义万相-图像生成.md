这份文档主要介绍的是阿里云**通义万相（Wanx）**大模型的**文本生成图像（Text-to-Image）**能力。

---

### 1. 核心功能简介
该 API 允许开发者通过输入文本描述（Prompt），调用通义万相大模型生成高质量的图像。它支持中英文输入，并提供多种艺术风格和分辨率选择。

*   **模型名称 (Model Name):** `wanx-v1`
*   **主要能力:** 文本生成图片。
*   **计费模式:** 通常按照生成的图片张数计费（具体价格需参考阿里云控制台）。

### 2. API 调用方式
该接口支持 **HTTP 调用** 和 **官方 SDK（Python/Java）** 两种方式。推荐使用 SDK，开发效率更高。

#### 任务模式：异步调用 (Asynchronous)
由于图像生成耗时较长（通常几秒到十几秒），该 API 采用**异步任务**机制：
1.  **提交任务:** 发送生成请求，服务器返回一个 `task_id`。
2.  **查询结果:** 使用 `task_id` 轮询查询接口，直到任务状态变为 `SUCCEEDED`，获取图片下载链接。

---

### 3. 关键参数详解 (Request Parameters)

在提交生成任务时，以下是核心参数：

| 参数名 | 类型 | 必填 | 说明 |
| :--- | :--- | :--- | :--- |
| **model** | String | 是 | 固定值：`wanx-v1` |
| **input.prompt** | String | 是 | 提示词，支持中英文。例如：“一只在太空漫步的猫，赛博朋克风格” |
| **input.negative_prompt** | String | 否 | 反向提示词（不希望出现的元素），例如：“低质量，模糊，变形” |
| **parameters.style** | String | 否 | 图像风格。可选值包括：`<auto>`(默认), `photography`(摄影), `portrait`(人像), `3d cartoon`(3D卡通), `anime`(二次元), `oil painting`(油画), `watercolor`(水彩), `sketch`(素描), `chinese painting`(国画), `flat illustration`(扁平插画) 等。 |
| **parameters.size** | String | 否 | 分辨率。支持：`1024*1024`, `720*1280`, `1280*720`。默认为 `1024*1024`。 |
| **parameters.n** | Integer | 否 | 生成数量，范围 1-4，默认 1。 |
| **parameters.seed** | Integer | 否 | 随机种子（0-4294967290），用于固定生成结果的随机性。 |

### 4. 代码示例 (Python SDK)

这是最快捷的集成方式。首先需要安装 DashScope SDK：
```bash
pip install dashscope
```

**Python 完整代码示例：**

```python
import dashscope
from dashscope import ImageSynthesis
from http import HTTPStatus

# 1. 设置 API KEY (建议在环境变量中配置，不要硬编码)
# export DASHSCOPE_API_KEY="sk-..."
dashscope.api_key = 'YOUR_DASHSCOPE_API_KEY'

def generate_image():
    # 2. 提交生成任务
    prompt = "一只穿着宇航服的柯基犬，站在火星表面，背景是地球，4k高清，电影质感"
    
    try:
        rsp = ImageSynthesis.call(
            model='wanx-v1',
            prompt=prompt,
            style='photography',  # 风格：摄影
            size='1024*1024',     # 分辨率
            n=1                   # 生成张数
        )
        
        # 3. 处理响应
        if rsp.status_code == HTTPStatus.OK:
            # 打印结果，通常包含 output.task_id (如果异步) 或 output.results (如果同步返回)
            # 通义万相 SDK 封装了轮询逻辑，直接返回结果链接
            if rsp.output and rsp.output.results:
                for result in rsp.output.results:
                    print(f"生成成功，图片URL: {result.url}")
            else:
                print(f"任务提交成功，任务ID: {rsp.output.task_id}")
        else:
            print(f"请求失败: code={rsp.code}, message={rsp.message}")
            
    except Exception as e:
        print(f"发生异常: {e}")

if __name__ == '__main__':
    generate_image()
```

### 5. HTTP 调用示例 (供参考)

如果不使用 SDK，可以直接发送 HTTP POST 请求。

**请求 URL:**
`https://dashscope.aliyuncs.com/api/v1/services/aigc/text2image/generation`

**Headers:**
*   `Authorization`: `Bearer YOUR_API_KEY`
*   `Content-Type`: `application/json`
*   `X-DashScope-Async`: `enable` (开启异步处理)

**Body (JSON):**
```json
{
    "model": "wanx-v1",
    "input": {
        "prompt": "水墨画风格的山水图，云雾缭绕"
    },
    "parameters": {
        "style": "chinese painting",
        "size": "1280*720",
        "n": 1
    }
}
```

### 6. 使用建议与注意事项

1.  **Prompt 优化:** 通义万相对中文理解能力很强，但也支持英文。描述越具体（包含主体、环境、光照、风格、视角），效果通常越好。
2.  **图片有效期:** 生成的图片 URL 是临时链接（通常有效期为 24 小时），请务必在生成后及时下载保存到自己的服务器或 OSS。
3.  **内容安全:** API 内置了安全审核，如果 Prompt 或生成的图片包含违规内容（色情、暴力、政治敏感等），任务会失败并返回相应的错误码。
4.  **错误处理:** 注意捕获异常，特别是网络超时或 API 限流（QPS 限制）。

### 总结
阿里云通义万相（Wanx）API 是一个成熟的商业级文生图接口，适合用于**广告素材生成、插画制作、游戏资产概念设计**等场景。通过 `wanx-v1` 模型和简单的 Python SDK 即可快速集成。
